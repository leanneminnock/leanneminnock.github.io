Bio

Hi, I'm Leanne. I am an aspiring full stack developer (and avid crocheter).
I have taken an opportunity in the last year to follow my interest in software development and hone my full stack skills. After many eureka moments and just a few exesential crises, I am finally at a stage where I can say that I am a developer.
So, let me tell you a little about my journey over the last year:

Features

Pokedex

Features
This app is a frontend practice app, using an external API. The focus was to practice routing, using Bootstrap and integrating with the Pokemon API using their documentation as a guide.
Challenges
Working with an external API was tricky at first, reading the docs was challenging until I figured out how the API worked. I ran into an obsatcle when I tried to integrate a search option into the site. After a lot of trial and error I deduced that the API didn't offer search capabilities. Now when looking into using an API I'm conscious of the possible limitations.

Dutch Treat

Features
This is a full stack application from database to site created as part of a Pluralsight course using a C# Dotnet Core back end and Angular front end.
An API was created to populate and query an SQL database using Entity Framework Core. The combination of razor pages and Angular was used to create a store front including a checkout page.
Challenges
There were many obstacles throughout this project, the most frustrating being the library versions in the course were out of date. This fundamentaly broke the application on numerous occasions and I had to do a lot of research to fix the breaking changes between versions. Part of the course taught about using auth tokens for user login varification. This I found tricky and still do as security is complex and has a high learning curve.
